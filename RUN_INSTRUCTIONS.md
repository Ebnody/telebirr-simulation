# How to Run the Telebirr Simulation

## 🚀 Option 1: Python Server (Already Running)
Open your browser and go to:
**http://localhost:8000**

## 🌐 Option 2: Open File Directly
1. Navigate to: `c:/Users/admin/CascadeProjects/windsurf-project/`
2. Double-click on `index.html`

## 🔧 Option 3: Different Server Methods

### Using Node.js (if installed):
```bash
cd c:/Users/admin/CascadeProjects/windsurf-project/
npx http-server
```
Then open: http://localhost:8080

### Using PHP (if installed):
```bash
cd c:/Users/admin/CascadeProjects/windsurf-project/
php -S localhost:8000
```

### Using Live Server in VS Code:
1. Open the project folder in VS Code
2. Install "Live Server" extension
3. Right-click on `index.html`
4. Select "Open with Live Server"
